# Sufaraa v9.6.64 - Audience Screen Rebuild

تم بناء شاشة الجمهور من جديد فوق نسخة v9.6.63 التي صفّرت الجمهور، مع الحفاظ على النسخة الآمنة v9.6.62 لمنطق المتسابق والميسر.

## ما تم بناؤه

- شاشة بداية قبل بدء اللعبة: شعار كبير + جملة "لم تبدأ المسابقة بعد" + الفرق المسجلة.
- أثناء المرحلة الأولى: ترتيب مباشر حسب نقاط المرحلة الأولى فقط.
- عند انتهاء المرحلة الأولى: عد تنازلي 5 إلى 0 ثم عرض النتائج العامة.
- أثناء المرحلة الثانية: ترتيب مباشر حسب نقاط المرحلة الثانية فقط.
- عند انتهاء المرحلة الثانية: عد تنازلي 5 إلى 0 ثم عرض النتائج العامة.
- أثناء المرحلة الثالثة: جدول الأسئلة + الدور الحالي + السؤال المختار + مؤقت الإجابة + إجابات الفرق مع العلامات عند reveal.
- عند انتهاء المرحلة الثالثة: عد تنازلي 5 إلى 0 ثم عرض النتائج العامة.
- أثناء المرحلة الرابعة: تظهر شاشة انتظار السؤال، ثم السؤال عند أمر الميسر، ثم مؤقت الإجابة، ثم إجابات الفرق مع العلامات والمتتالية عند reveal.
- عند انتهاء المرحلة الرابعة أو عرض النتائج النهائية: عد تنازلي 5 إلى 0 ثم النتائج العامة، وبعدها بوديوم للفرق الثلاثة الأولى.

## مصادر البيانات المعتمدة

- `meta/gameFlow` لمسار المسابقة.
- `teams` لنقاط الفرق وترتيبها.
- `meta/stage3Final` لحالة المرحلة الثالثة.
- `meta/stage4Final` لحالة المرحلة الرابعة.

## ملاحظة مهمة

لم يتم لمس منطق المتسابق أو لوحة الميسر. التعديل محصور في شاشة الجمهور وملفاتها فقط.
