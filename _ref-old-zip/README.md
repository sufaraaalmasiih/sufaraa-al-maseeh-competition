## أهم التعديلات

## V9.4 Refined Gameplay & Judge Control
- يمكن تعديل اسم الفريق وأسماء اللاعبين من واجهة اللاعبين ولوحة الميسر.

- تمت إضافة خيار ترحيل كل الفرق إلى مرحلة معينة.

## V9.4.7 - المرحلة الأولى والوسائط

### المرحلة الأولى
تم تحديث المرحلة الأولى لتعرض أنواع الأسئلة الأربعة بشكل عشوائي داخل نفس الجولة:
- ماذا ينقص
- اختر من متعدد
- رتّب
- فراغات

يبقى النظام محافظًا على 5 دقائق، 50 سؤالًا كحد أقصى، و5 نقاط للإجابة الصحيحة بدون إظهار الصح أو الخطأ للمتسابق.

### رفع الصور والفيديوهات
تم تجهيز مجلدات الوسائط داخل المشروع:

```text
assets/images/
assets/videos/
```

لإضافة صورة أو فيديو ثابت:
1. ضع الصورة داخل `assets/images` أو الفيديو داخل `assets/videos`.
2. استخدم المسار في HTML أو داخل بيانات الأسئلة، مثل:
   - `assets/images/question-1.jpg`
   - `assets/videos/intro-stage1.mp4`
3. ارفع المشروع على GitHub Pages كالمعتاد.

ملاحظة: إذا أردت رفع الصور والفيديوهات من لوحة التحكم نفسها لاحقًا، نحتاج تفعيل Firebase Storage وإضافة قسم إدارة وسائط في لوحة الميسر.
