# الاختبارات

## 1) اختبارات الوحدة (المنطق) — `npm test`
108 اختباراً تغطّي المنطق الحتمي (الترتيب، التعادل، النقاط، توزيع المرحلة 3 والأسئلة
الجماعية، مرونة Excel، المحاكاة بأعداد فرق مختلفة). **لا تحتاج أي إعداد** وتعمل بسرعة.

## 2) اختبارات أمان قواعد Firestore — `npm run test:rules`
تتحقق آلياً من `firestore.rules` عبر **Firebase Emulator محلي** (لا تستهلك أي حصة من
الباقة المجانية). تغطّي:
- منع فريق من قراءة/تعديل بيانات فريق آخر.
- حدود النقاط (مكافحة التلاعب: pointsDelta بين −200 و+25).
- خصوصية أرشيف الفريق (`teamArchives`) — الفريق يرى أرشيفه فقط.
- الاعتراضات (`objections`) — الفريق يُنشئ/يقرأ اعتراضه فقط، الميسّر يحكّم.
- السجل (`history`) — للميسّر فقط.
- `gameFlow` — قراءة عامة، كتابة محمية للميسّر.

### المتطلّبات لتشغيلها
- **JDK 21 أو أحدث** (الـ Emulator الحالي يتطلب Java 21+).
  - تحقّق: `java -version`. حمّل من: https://adoptium.net
- `firebase-tools` يُنزّل تلقائياً عبر `npx` عند أول تشغيل.

### التشغيل
```bash
npm run test:rules
```
يشغّل الأمر الـ Emulator، ينفّذ اختبارات القواعد، ثم يُغلقه تلقائياً.

> ملاحظة: لم تُشغَّل في بيئة التطوير الحالية لأن Java فيها 1.8 (أقدم من المطلوب).
> الاختبارات مكتوبة وجاهزة — شغّلها على جهازك بعد تثبيت JDK 21+.
